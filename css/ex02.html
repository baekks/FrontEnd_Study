스타일 지정 방식

  • Inline Style(인라인 스타일 지정 방식) : 해당 요소에 style 속성으로 직접 스타일을 지정
  • Internal Style(내부 스타일 지정 방식) : 현재 문서에 style 태그 내에 선택자를 이용하여 스타일을 부여하는 방식
  • External Style(외부 스타일 지정 방식) : 별ㄷ의 외부 스타일 문서(CSS 파일)를 link로 연결하여 스타일을 지정하는 방식

선택자(Selector)

스타일을 부여할 대상을 의미하며, 아이디, 클래스, 태그, 계층 등의 정보를 활용해 선택하여 스타일을 부여할 수 있게 한다.

기본 선택자

  • * : 전체 선택자로 스타일 부여할 요소가 모든 태그
  • 태그(element) : 해당 태그를 선택하여 스타일 부여
  • 클래스(class) : .(점)을 구분자로 하는 선택자로 같은 스타일을 적용
  • 아이디(id) : #(샵)을 구분자로 하는 선택자로 해당 문서에 유일하게 있는 요소에 스타일 적용

아이디 > 클래스 > 태그 순으로 우선 순위가 있으며, 선택자가 같고, 속성이 같은 경우 마지막에 설정한 속성이 적용됨.

계층 선택자

  • 부모 / 자식 선택자 : > 를 구분자로 하여 앞에는 부모 요소와 뒤에는 자식 요소를 나타내며, 특정 부모 요소에 속하는 자식 요소를 선택하여 스타일 적용

    우선순위
    # : 금메달   . : 은메달   태그 : 동메달
    .arc > .menu > li 은2 동1
    div.arc > ul.meun > li 은2 동3
    div > ul > li 동3
    아래처럼 큰 요소에서 작은 요소로 차례대로 스타일 지정하여 부여
    div
    div > ul
    div > ul > li

  • 조상 / 후손 선택자 : 조상 요소와 후손 요소 사이에 한 칸의 공백으로 구분하여 특정 요소 안에 있는 후손 요소를 선택하여 스타일 적용할 때

    .menu li
    .arc li
    위와 같이 띄워쓰기로 조상과 후손을 구분하여, 특정 조상의 후손을 선택

  • 복수 선택자 : 여러 요소를 선택할 때 콤마(,)를 이용하여 동시에 같은 스타일을 부여할 때, 한꺼번에 여러 요소를 선택함

    p, li, div
    .menu, li, a
    #menu, li.first
    위와 같은 한 번에 여러 요소를 선택 => 복수 선택자

  • 순서 / 위치 선택자

    .menu li:first-child : 첫 번째 자식
    .menu li:nth-child(2) : 두 번째 자식
    .menu li:nth-child(2n) : 짝수 번째 자식
    .menu li:nth-child(2n+1): 홀수 번째 자식
    .menu li:nth-child(3)
    .menu li:last-child : 마지막 자식
    child 개념은 현재 그룹내의 순서/위치로 구분하여 선택
    :(콜른) 앞의 요소 중에 해당 순서에 맞는 요소를 선택

    .menu a:first-of-type: 첫 번째 요소
    .menu a:nth-of-type(2): 두 번째 요소
    .menu a:nth-of-type(2n): 짝수 번째 요소
    .menu a:nth-of-type(2n+1): 홀수 번째 요소
    .menu a:last-of-type: 막내 요소
    .menu a:only-of-type: 단독 요소
    of-type 개념은 현재 그룹 내의 요소와 순서가 모두 일치하면 선택
    순서를 먼저 판단하고, :(콜른) 앞의 해당 요소를 선택
