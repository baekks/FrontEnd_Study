CSS(Cascading Style Sheet)

상위에서 하위까지 단계적으로 HTML의 Style을 지정하여 적용하는 것

구조와 표현의 분리

HTML은 뼈대와 같은 구조이며, CSS는 뼈대에 살을 입히면서 원하는 디자인을 표현하는 것을 말한다.

Style의 기본 문법

선택자 { 속성1 : 값1; 속성2 : 값2; ...; 속성n : 값n; }

CSS 구성

  • 선택자(Selector) : 스타일을 부여할 대상
  • 그룹핑 요소(Separater = Grouping) : { }로 선택자에 의해 한 번에 부여되는 스타일 묶음
  • 속성(Properties) : 모양, 크기, 색깔 등의 특성
  • 값(Values) : 속성으로 지정 가능한 내용
  • 단위(Unit) : 크기, 색상, 모양, 위치의 단위로 px, %, pt, em, vw, vh
