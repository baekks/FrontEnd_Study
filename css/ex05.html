CSS 기본 속성

CSS 글자/문단 관련 속성

  • color: 글자색으로 컬러명, HEX, RGB(RGBA), HSL(HSLA) 단위로 지정 가능
  • font-size: 글자 크기로 px, pt, em, % 등의 단위로 지정 가능
  • font-family: 글꼴로 해당 글꼴이 서버 또는 컴퓨터에 설치가 되어 있거나 또는 웹 폰트를 지정하여 사용할 수 있음
  • font-weight: 글자 두께로 nomal(기본값), light, bold, 100~900 100 단위씩 글꼴에 따라 지정이 가능
  • line-height: 줄 간격으로 px, pt, em 단위 등을 사용할 수 있지만, 보통 생략하여 em 단위로 지정
  • text-align: 글자 정렬 방식으로 left(기본값), center, right 등을 지정
  • text-decoration: 글자 장식으로 underline(밑줄), overline(윗줄), through(취소선) 등을 지정
  • text-shadow: 글자에 그림자를 넣는 것으로 (x량 y량 blur량 그림자색) 등을 지정
