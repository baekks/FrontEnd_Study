속성 선택자

  • [attr] : 해당 속성이 있으면 선택, ex) a[href]
  • [attr^=문자열] : 해당 속성이 문자열로 시작하면 선택, ex) img[src^=mb]
  • [attr$=문자열] : 해당 속성이 문자열로 끝나면 선택, ex) img[src$=mb]
  • [attr*=문자열] : 해당 속성에 문자열이 포함되면 선택, ex) img[src*=mb]

예를 들어, img[src], img[src^="mb"], img[src$="png"], img[src*="03"] 등이 있음.

링크/반응 선택자

  • :link - a 요소에 href(링크)가 있는 요소
  • :visited - a 요소에 있는 href 속성 값을 방문한 적이 있는 요소
  • :hover - 해당 요소에 마우스 포인터를 올렸을 때 선택
  • :active - 해당 요소에 마우스를 클릭했을 때 선택

링크/반응 선택자는 a 요소에 적용하는 선택자이지만, :hover 와 :active는 다른 태그에도 사용이 가능합니다.

상태 선택자

  • :checked - 현재 상태가 check 된 상태이면, 선택되는 것으로 체크박스(checkbox)나 라디오(Radio) 버튼에 사용할 수 있음
  • :enabled - 현재 요소가 사용이 가능한 상태이면, 선택하는 것으로 타입(input)에 관계없이 input 요소에서 사용이 가능한 요소를 선택할 때 사용
  • :disabled - 현재 요소가 사용이 불가능한 상태이면, 선택하는 것으로 타입(input)에 관계없이 input 요소에서 사용이 불가능한 요소를 선택할 때 사용 (전송 안됨)

상태 선택자는 주로 폼 요소가 어떠한 상태인지에 따라 선택하여 스타일 적용




이중 반응 선택자

  • ::first-letter - 해당 요소의 첫 글자만 선택
  • ::first=line - 해당 요소의 첫 줄만 선택
  • ::selection - 마우스로 text의 범위를 선택
  • ::before - 특정 요소의 앞을 선택
  • ::after - 특정 요소의 뒤를 선택

위 다섯 가지는 가상 선택자로 실제 선택이나 탐색이 불가능합니다.
