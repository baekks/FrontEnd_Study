배치 관련 속성

위치(position)

position: 위치 기준을 지정하는 속성. static|relative|absolute|fixed|sticky

  • static: 정적. width, height, margin, float, clear 속성을 활용하여 배치
  • absolute: 절대. left/right, top/bottom, z-index 속성을 활용하여 배치
  • relative: 상대. width, height, margin, float, clear 속성을 활용하여 배치
  • fixed: 화면고정 위치. left/right, top/bottom, z-index 속성을 활용하여 배치

흐름 속성

float: 흐름(부유) 속성. left|right

흐름 해제 속성

clear: 흐름(float) 속성 해제. left|right|both

좌표 속성

  • left: 왼쪽으로 부터 기준으로 위치 설정. px, % 등 단위와 auto
  • right: 오른쪽으로 부터 기준으로 위치 설정. px, % 등 단위와 auto
  • top: 꼭대기로 부터 기준으로 위치 설정. px, % 등 단위와 auto
  • bottom: 바닥으로 부터 기준으로 위치 설정. px, % 등 단위와 auto
  • left/right 중에서 하나만 사용해야 되지만, 가끔 left가 설정되어 있는 상태에서 right를 설정하면, left를 auto로 설정하면, right 적용됨
