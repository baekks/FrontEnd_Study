박스 관련 속성

inline(인라인 방식 태그): a, span, strong, em, ins, input, button => 크기 지정이 불가능하고, 글자 정렬X, 상/하 margin X, 줄 간격 지정 X -> 기본적으로 한 줄로 배치

inline-block(인라인 블록 태그): img, video, audio, => 크기 지정이 가능하지만, 한 줄로 배치됨

block(블록 방식 태그): div, header, footer, main, nav, aside, figure, section, article, h1~h6, .... => 크기 지정이 가능하고, 한 줄에 하나만 배치

크기

링크
  • width: 폭. auto, px, %, vw 단위
  • height: 높이. auto, px, %, vh 단위

바깥여백

  • margin: 바깥여백을 한 번에 지정. 10px, 10px 20px, 10px 20px 30px 40px
  • margin-top: 바깥 위 여백. px, % 단위
  • margin-right: 바깥 오른쪽 여백. px, % 단위
  • margin-bottom: 바깥 아래 여백. px, % 단위
  • margin-left: 바깥 왼쪽 여백. px, % 단위

안 여백

  • padding: 안 여백을 한 번에 지정. 10px, 10px 20px, 10px 20px 30px 40px
  • padding-top: 안 위 여백. px, % 단위
  • padding-right: 안 오른쪽 여백. px, % 단위
  • padding-bottom: 안 아래 여백. px, % 단위
  • padding-left: 안 왼쪽 여백. px, % 단위

테두리(border, outline)

  • border: 테두리 통합 속성으로 상/우/하/좌의 선두께 선모양 선색
  • border-width: 선 두께를 지정하는 세부 속성. px 단위
  • border-style: 선 모양을 지정하는 세부 속성. solid(실선)|dashed|dotted|hidden|double
  • border-color: 선 색을 지정하는 세부 속성. 컬러명, HEX, rgb(rgba), hsl(hsla)
  • border-top: 상단 테두리의 선두께 서모양 선색을 지정하는 세부 속성
  • border-right: 우측 테두리의 선두께 서모양 선색을 지정하는 세부 속성
  • border-bottom: 하단 테두리의 선두께 서모양 선색을 지정하는 세부 속성
  • border-left: 좌측 테두리의 선두께 서모양 선색을 지정하는 세부 속성
  • border-[방향]-width: 특정 방향(상우하좌)의 선 두께를 지정하는 세세 속성
  • border-[방향]-style: 특정 방향(상우하좌)의 선 모양을 지정하는 세세 속성
  • border-[방향]-color: 특정 방향(상우하좌)의 선 색을 지정하는 세세 속성
  • 스타일을 지정할 때 통합속성 -> 세부속성 -> 세세속성 순으로 지정해야함

모서리 둥글기

  • border-radius: 모서리의 둥글기를 지정하는 세부속성. px,% 단위 10px, 10px 20px, 10px 20px 30px, 10px 20px 30px 40px. lt/rb rt/lb lt rt rb lb
