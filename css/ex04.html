CSS 속성 단위

CSS의 속성을 지정할 때 속성의 값 뒤에 붙는 단위를 말함.
ex) 20px, 10mm, 100% ... 에서 px, m, % 등

길이/위치 단위

크기나 위치의 값을 숫자 뒤에 붙이는 단위로 절대 단위와 상대 단위가 있다.

  • 절대 단위 : px, pt, cm, mm, in, inch, ..
  • 상대 단위 : %, em(비율=배율), rem(반응형), vw, vh, ch(폭),

색상 단위

색상 단위는 색량(채도)이나 명도, 책의 크기를 지정하는 방식

  • 컬러명 - color : black
  • HEX(16진수) 6자리 - color : #2c82c8
  • HEX(16진수) 3자리 - color : #f00
  • RGB - rgb(0~255, 0~255, 0~255) - color : rgb(255,0,0)
  • RGBA - rgba(0~255, 0~255, 0~255, 0~1(실수가 들어가고 투명도 조절)) - color : rgba(255,0,0,0.5)
  • HSL - hsl(hue,saturate,lightness) - color : hsl(0,50%, 100%)
  • HSLA - hsla(hue,saturate,lightness) - color : hsl(0,50%, 100%,0.5)

색상(Color) : 색(Hue) + 채도(Saturation) + 명도(Lightness/Brightness)
alpha(=opacity) : 불투명도 0~1

시간 단위

시간 단위는 애니메이션이나 트랜지션 등에서 사용되는 단위로 s, ms 등을 활용
ex) transition: 1s

각도 단위

각도 단위는 변형에서 방향이나 회전시에 사용되는 단위로 deg 등을 활용
ex) transform: 45deg
